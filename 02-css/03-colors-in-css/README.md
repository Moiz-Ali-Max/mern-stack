# Colors in CSS
Colors in CSS can be specified by the following methods:
1. Hexadecimal Colors
2. RGB Colors
3. Predefined / Cross-browser color names
4. RGBA Colors
5. HSL Colors
6. HSLA Colors

### 1. Hexadecimal Colors
A hexadecimal colors is specified with #RRGGBB, where RR(Red), GG(Green), BB(Blue).

### 2. RGB Colors
An RGB Color value is specified with rgb() function, which has this syntax: rgb(red, green, blue) -> rgb(0, 255, 255).

### 3.Pre-defined / Cross Browser Color Names
140 color names are predefined in the HTML and CSS color specification.
